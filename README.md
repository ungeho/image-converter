# Image Converter

React + Vite で作成した、ブラウザ上で画像形式を変換できるアプリです。  
ドラッグ＆ドロップ、ファイル選択、クリップボード貼り付けに対応し、画像をすばやく別形式へ変換してダウンロードできます。  
複数画像の一括変換、リサイズ、圧縮前後のサイズ比較、変換画像のクリップボード再コピー、ZIP 一括保存にも対応しています。

## 概要

このアプリは、ローカル画像をブラウザ上で読み込み、`canvas` を使って別の画像形式へ変換するシンプルなツールです。  
サーバーアップロードを前提とせず、フロントエンドだけで完結する構成になっています。

## 主な機能

- 画像ファイルのドラッグ＆ドロップ
- 画像ファイルの手動選択
- クリップボード内画像の貼り付け
- 複数画像の一括読み込みと一括変換
- PNG / JPEG / WebP / AVIF への変換
- 変換前・変換後プレビュー表示
- 変換後ファイルのダウンロード
- 変換済み画像の一括ダウンロード
- 変換済み画像の ZIP 一括保存
- JPEG / WebP 用の品質調整
- リサイズ指定と縦横比の維持 / 解除
- 圧縮前後のファイルサイズ比較
- 変換後画像のクリップボード再コピー
- モバイル幅でも崩れにくいレスポンシブUI

## 対応形式

入力:

- ブラウザが読み込める画像ファイル
- クリップボード上の画像データ

出力:

- PNG
- JPEG
- WebP
- AVIF

## 仕様

- 画像変換はブラウザの `canvas` を使って実行します。
- サーバー側の画像処理は行いません。
- PNG は可逆圧縮のため、品質スライダーの値は変換結果に反映されません。
- JPEG は透過を保持できないため、透明部分は白背景で出力します。
- AVIF はブラウザが出力エンコードに対応している場合のみ選択肢に表示されます。
- クリップボード貼り付けは、ブラウザやOSの仕様によって利用可否が変わる場合があります。
- 変換後画像のクリップボードコピーは、対応ブラウザでのみ利用できます。
- ZIP 一括保存では、変換済み画像を1つの `.zip` ファイルにまとめてダウンロードできます。
- リサイズ有効時は、幅と高さの指定内容に応じて出力解像度が変化します。
- 縦横比維持を有効にした場合、指定サイズの枠に収まるように画像を縮放します。
- AVIF の利用可否はブラウザ実装差があるため、実行環境に応じて自動判定します。

## 使い方

### 画像ファイルを変換する

1. アプリを開きます。
2. 画像をドラッグ＆ドロップするか、`画像を選択` ボタンから画像を読み込みます。
3. `出力形式` を選びます。
4. 必要に応じて品質やリサイズ設定を調整します。
5. 一覧から確認したい画像を選びます。
6. 変換完了後、個別ダウンロード、一括ダウンロード、または ZIP 保存で保存します。

### クリップボード画像を変換する

1. スクリーンショットなどで画像をクリップボードにコピーします。
2. アプリ画面をアクティブにします。
3. `Ctrl + V` で貼り付けます。
4. 画像が読み込まれたら、出力形式を選んでダウンロードします。

### 変換画像をクリップボードにコピーする

1. 一覧から対象画像を選びます。
2. 変換完了後、`変換後をクリップボードにコピー` を押します。
3. 対応アプリにそのまま貼り付けます。
