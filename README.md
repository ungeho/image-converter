# Image Converter

React + Vite で作成した、ブラウザ上で画像形式を変換できるアプリです。  
ドラッグ＆ドロップ、ファイル選択、クリップボード貼り付けに対応し、画像をすばやく別形式へ変換してダウンロードできます。

## 概要

このアプリは、ローカル画像をブラウザ上で読み込み、`canvas` を使って別の画像形式へ変換するシンプルなツールです。  
サーバーアップロードを前提とせず、フロントエンドだけで完結する構成になっています。

## 主な機能

- 画像ファイルのドラッグ＆ドロップ
- 画像ファイルの手動選択
- クリップボード内画像の貼り付け
- PNG / JPEG / WebP への変換
- 変換前・変換後プレビュー表示
- 変換後ファイルのダウンロード
- JPEG / WebP 用の品質調整
- モバイル幅でも崩れにくいレスポンシブUI

## 対応形式

入力:

- ブラウザが読み込める画像ファイル
- クリップボード上の画像データ

出力:

- PNG
- JPEG
- WebP

## 仕様

- 画像変換はブラウザの `canvas` を使って実行します。
- サーバー側の画像処理は行いません。
- PNG は可逆圧縮のため、品質スライダーの値は変換結果に反映されません。
- JPEG は透過を保持できないため、透明部分は白背景で出力します。
- クリップボード貼り付けは、ブラウザやOSの仕様によって利用可否が変わる場合があります。
- 対応形式は、主要ブラウザで安定して扱いやすい PNG / JPEG / WebP に絞っています。

## 使い方

### 画像ファイルを変換する

1. アプリを開きます。
2. 画像をドラッグ＆ドロップするか、`画像を選択` ボタンから画像を読み込みます。
3. `出力形式` を選びます。
4. 必要に応じて品質を調整します。
5. 変換完了後、ダウンロードボタンから保存します。

### クリップボード画像を変換する

1. スクリーンショットなどで画像をクリップボードにコピーします。
2. アプリ画面をアクティブにします。
3. `Ctrl + V` で貼り付けます。
4. 画像が読み込まれたら、出力形式を選んでダウンロードします。
